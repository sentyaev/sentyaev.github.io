Configure Python virtual environment

Sat, Nov 25, 2017

Create Virtual Environment

Assume you’d like to create venv in ~/dev/env/py3

mkdir -p ~/dev/env/py3
python3 -m venv ~/dev/env/py3

To activate venv

source ~/dev/env/py3/bin/activate

To deactivate

deactivate
